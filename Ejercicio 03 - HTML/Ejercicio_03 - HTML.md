# Ejercicio 03 - HTML

## Introducción

El **HTML** (*HyperText Markup Language*) es el lenguaje estándar para crear páginas web. Define la estructura y el contenido de un documento mediante **etiquetas**, que indican al navegador cómo mostrar cada elemento: títulos, párrafos, imágenes, enlaces, formularios, entre otros.

## Síntesis

Un documento HTML se estructura con etiquetas que se abren y cierran, como `<p>texto</p>`. Su estructura básica siempre incluye:

- `<!DOCTYPE html>` — declara el tipo de documento
- `<html>` — elemento raíz del documento
- `<head>` — contiene metadatos (título, charset, estilos)
- `<body>` — contiene el contenido visible de la página

Las etiquetas se dividen en dos tipos: **de bloque** (como `<div>`, `<p>`, `<h1>`) que ocupan toda la línea, y **en línea** (como `<span>`, `<a>`, `<img>`) que ocupan solo su contenido. Los atributos como `class`, `id`, `src` y `href` permiten agregar información extra a los elementos.

## Reflexión

Lo que me parece más útil de HTML es que es el punto de partida de todo en la web. Sin importar qué tan compleja sea una aplicación, por debajo siempre hay HTML. Aprender a escribirlo correctamente —usando etiquetas semánticas como `<header>`, `<nav>`, `<main>` y `<footer>`— no solo organiza mejor el código, sino que también mejora la accesibilidad y el posicionamiento en buscadores. Es un lenguaje simple de aprender pero con mucho impacto.

---

## Conclusión

HTML es la base fundamental del desarrollo web. Conocer su estructura, sus etiquetas y sus atributos es el primer paso para construir cualquier página o aplicación web. Combinado con CSS y JavaScript, forma la triada esencial que todo desarrollador web debe dominar.
