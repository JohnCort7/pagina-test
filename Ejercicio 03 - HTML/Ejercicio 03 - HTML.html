Ejercicio 03 - HTML

Introducción

El HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura y el contenido de un documento mediante etiquetas, que le indican al navegador cómo mostrar cada elemento: títulos, párrafos, imágenes, enlaces, entre otros.

Síntesis

Un documento HTML se estructura con etiquetas que se abren y cierran, como <p>texto</p>. Su estructura básica siempre incluye:

  • <!DOCTYPE html> — declara el tipo de documento
  • <html> — elemento raíz del documento
  • <head> — contiene metadatos como el título y el charset
  • <body> — contiene el contenido visible de la página

Las etiquetas se dividen en de bloque (como <div>, <p>) que ocupan toda la línea, y en línea (como <span>, <a>) que solo ocupan su contenido.

Reflexión

Lo que encuentro más útil de HTML es que es el punto de partida de todo en la web. Sin importar qué tan compleja sea una aplicación, por debajo siempre hay HTML. Aprender a usarlo correctamente con etiquetas semánticas mejora tanto la organización del código como la accesibilidad de la página.

Conclusión

HTML es la base fundamental del desarrollo web. Conocer su estructura, sus etiquetas y sus atributos es el primer paso para construir cualquier página o aplicación web. Combinado con CSS y JavaScript, forma la triada esencial que todo desarrollador debe dominar.
